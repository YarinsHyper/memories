# Memories — Experiences Showcase Website

A personal **memories and experiences showcase website** built to present meaningful moments, events, and life highlights in a clean and visual way.

This project focuses on layout design, user experience, and frontend structure using vanilla web technologies.

The website is a portfolio/demo project and not intended for reuse or redistribution.

## Overview

The Memories website presents a collection of personal experiences in a visually structured format.

It demonstrates how a static website can be organized to tell a story through images, text, and interaction while keeping the codebase simple and readable.

The goal of the project is to showcase:

* Frontend layout design
* DOM manipulation
* Interactive UI behavior
* Clean file organization
* Visual storytelling through web structure

---

## Features

* Visual gallery of experiences
* Structured content sections
* Interactive elements
* Responsive layout
* Lightweight and fast loading
* Simple navigation flow

---

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

No frameworks or libraries are required to run the project.
